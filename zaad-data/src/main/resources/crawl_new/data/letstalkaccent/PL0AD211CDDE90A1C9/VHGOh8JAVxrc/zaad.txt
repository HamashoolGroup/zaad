Vocabulary for esl learners, learn to speak fluent English Todays new word is " back to square one" Back to square one , this is an idiom used to describe some activity that failed, and you had to start working on it from the beginning so, If you are back to square one, it means, you have to start from the beginning again. In short, you have to stop and start again, usually because something isn't working as well as expected.
