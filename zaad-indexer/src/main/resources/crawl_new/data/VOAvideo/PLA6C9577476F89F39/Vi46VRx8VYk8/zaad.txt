Ugandan Tree Bark Cloth Wows Europe's Fashion Runway
Published on Aug 8, 2013
Tree bark cloth from Uganda is being used to create some of the latest runway fashions in Europe. The technique used to make the bark into cloth is ancient, but a German-Nigerian designer is hoping it will help create a whole new fashion trend. VOA's Mary Alice Salinas reports.
